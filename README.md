## Installing
1. <code>git clone https://github.com/sebastienbruno/test-transdev-fullstack</code>
2. <code>cd test-transdev-fullstack</code>
3. <code>mvn install</code>
4. <code>java -jar target/online-bus-ticket-ws-<version>.jar</code>
